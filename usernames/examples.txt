Output a new username to System.out:

    java -cp classes edu.ucla.cens.awuser.CreateUserName data/username-words.txt 
    
Add a new user (verbose): 

    java -cp lib/*:classes edu.ucla.cens.awuser.ManageUser add data/add-remove-user.properties
    
Remove a user (quiet):

    java -cp lib/*:classes edu.ucla.cens.awuser.ManageUser quiet remove data/add-remove-user.properties


Create and edit your own copy of add-remove-user.properties for managing users.
